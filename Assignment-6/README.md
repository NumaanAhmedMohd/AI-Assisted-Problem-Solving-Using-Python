# Person class demo

This small example shows a `Person` class with a constructor and a `display_details()` method.

Files:
- `person.py` - contains the `Person` class.
- `run_demo.py` - demo script that constructs a `Person` and prints the details.

How to run:

Open a terminal in this folder and run:

```powershell
python .\run_demo.py
```
